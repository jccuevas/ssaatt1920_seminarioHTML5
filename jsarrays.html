Introducción a JavaScript - arrays

En este ejemplo se muestran ejemplos sencillos del manejo de arrays en JavaScript

Definición y acceso a los elementos

Definición de un array

<script>

let lista=["Macarrones","Espaguetis","Lazos","Tallarines"];
document.write("Un array:"+lista);

Acceso a elementos empleando array[...]

<script>

for (let i = 0; i < 4; i++) {
    document.write("Elemento " 
            + i 
            + ":" 
            + lista[i] 
            + "<br>");
}

Acceso a elementos for...of

<script>

for (const elemento of lista) {
    document.write("Elemento:" 
            + elemento 
            + "<br>");
}

Iterar y cambiar los elementos

<script>

let numeros=[1,2,3,4,5];
for (const elemento of numeros) {
    document.write("Elemento previo" + elemento + "<br>");
}
for (let i = 0; i < numeros.length; i++) {
    numeros[i]= numeros[i]*2;
}

for (const elemento of numeros) {
    document.write("Elemento nuevo:" + elemento + "<br>");
}
