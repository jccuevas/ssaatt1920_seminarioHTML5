Ejemplo de aplicación de estilos al texto

           div {
                border: 1px solid black;
                padding: 10px;
                width: 200px;
                height: 200px;
                text-align: justify;
            } 
           h1{
                text-align: center;
            }

Como se puede ver el encabezado está centrado y este texto justificado al estilo del que aparece en libros, periódicos y revistas.
