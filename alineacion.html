Ejemplo de aplicación estilos para alinear elementos

La alineación con position y float ya se ha visto en otros ejemplos

Centrar con márgenes

            .centro{
                margin: auto;
                width: 50%;
                min-width: 400px;
                background-color: lightsteelblue;
                color: white;
            }

Centrado

Un div de tamaño 50% centrado en la página

Centrados vertical con padding

            .vertical{
                margin: auto;
                padding: 50px 0;
                width: 50%;
                min-width: 400px;
                background-color: lightsteelblue;
                color: white;
            }

Centrado vertical

Un div de tamaño 50% centrado en la página
