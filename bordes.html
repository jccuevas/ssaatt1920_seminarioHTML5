Ejemplo de aplicación estilos para los bordes de un elemento

Como se ha visto, cada elemento se puede rodear por bordes entre el relleno y el margen. La propiedad principal que hace que se muestre un borde alrededor de un objeto es border-style.

Bordes sencillos

            .bordesencillo{
                width: 200px;
                
                border-style: solid;
                margin: 20px;
            }   
            .bordesdiferentes{
                width: 200px;          
                border-style: solid dotted solid dotted;
                margin: 20px;
            }      
            .completo{
                width: 200px;
                border-style: solid;
                border-width: 0.5em;
                border-color: red;
                margin: 20px;
            }
            .colores{
                width: 200px;
                border-style: dashed;
                border-width: 0.5em;
                border-color: red blue green violet;
                margin: 20px;
            }       
            .atajo{
                width: 200px;
                border: 0.5em solid red;
                margin: 20px;
            }
            .izquierda{
                padding: 2px;
                margin: 20px;
                border-left: 0.5em solid red;
                background-color: lightgray;
            }
