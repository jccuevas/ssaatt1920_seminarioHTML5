Búsqueda de elementos que no se válidos

Con este código, se puede relegar a HTML la validación, si ésta es suficiente, y comprobar dichos valores con JavaScript

Abre la consola

<script>

function buscarNoValidos() {
    let elementos = document.querySelectorAll("input:invalid");
    for (let node of elementos) {
        console.log("Elemento " + node.nodeName + " name=" + node.name + " del tipo " + node.type);
    }
}

Pulsa el botón para mostrar los elementos input no válidos
