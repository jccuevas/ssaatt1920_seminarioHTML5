Ejemplo de aplicación de estilos a fuentes de texto

Cambie el ancho de la ventana del navegador para ver cómo se adapta el tamaño de las fuentes. Esto se debe a que el tamaño de las fuentes se ha especificado con unidades relativas

            body{
                font-family: verdana;
                font-size: 2vw;
            }
            p.cursiva{
                font-style: italic;

            }
            p.capital::first-letter{
                font-size: 2em;
                color: red;
            }
            .b{
                font-weight: bold;
            }
            h1{
                font-size: 5vw;
            }
